# Projects

## Interactive Portfolio Globe

A 3D interactive wireframe globe that serves as the navigation hub for my portfolio website. Built with HTML Canvas and custom 3D rendering.

**Features:**
- Drag-to-rotate with momentum physics
- Clickable logo markers for different sections
- Zoom transitions when entering sections
- Inside-sphere background effect for immersive experience

**Technologies:** React, HTML Canvas, Custom 3D Math, CSS Animations

## Vectorization Insight Tool

An educational web application for visualizing and understanding the document vectorization process used in RAG (Retrieval-Augmented Generation) systems.

**Features:**
- Visual chunk boundary highlighting
- Real-time chunking configuration
- 2D/3D embedding visualization with UMAP/t-SNE
- Similarity matrix heatmaps
- Export to various vector database formats

**Technologies:** React, FastAPI, ChromaDB, Sentence-Transformers, Plotly
